First Steps

es una aplicación web que marca nuestros primeros pasos en el desarrollo con Next.js. El proyecto incluye una página de inicio (Home) y varias rutas para explorar. Durante su desarrollo, exploramos la diferencia entre Layout y Page en Next.js, aprendiendo cómo estructurar y organizar la aplicación. El enfoque principal del proyecto fue educativo, permitiéndonos familiarizarnos con los conceptos básicos y las primeras interacciones con los datos.

My Dashboard

es una aplicación web que permite explorar la información de Pokémon de manera interactiva. Utiliza Next.js para el frontend y se conecta a una API externa para obtener los datos de los Pokémon. Los usuarios pueden dar "likes" a los Pokémon, y el estado se gestiona con Redux Toolkit y se guarda en LocalStorage para persistir la información. La app también usa rutas en Next.js para mostrar detalles adicionales de cada Pokémon.

Admin Todos

es una aplicación de gestión de tareas que permite a los usuarios crear, editar y eliminar tareas. El backend está construido con Next.js y Prisma, usando PostgreSQL como base de datos. La autenticación se maneja con Auth.js, permitiendo iniciar sesión con GitHub y Google. La aplicación está desplegada en Docker, lo que facilita su escalabilidad.

Teslo Shop

una réplica del sitio de Tesla, utilizando Next.js para el frontend, Prisma con PostgreSQL, Docker para contenedorización, y Auth.js para la autenticación. Implementamos un sistema de pagos con PayPal SDK y gestionamos el estado de la aplicación con Zustand. Las funcionalidades clave incluyen la gestión de productos, carrito de compras y pago, con un enfoque en la experiencia del usuario y el rendimiento.
